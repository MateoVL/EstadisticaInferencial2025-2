Estamos trabajando con datos independientes y debemos ver diferencias significativas en la eficiencia entre las muestras B y C, que podra expresarse como la media de los tiempos de ejecucin. Por lo tanto para responder las dudas de la memorista podramos aplicar una prueba t de Student para muestras independientes.

Vamos a verificar las condiciones para esta prueba:

  1. Las observaciones son independientes entre s: Las muestras se seleccionaron de forma aleatoria e independiente entre s a partir de una poblacin con datos independientes, ya que se tratan de ejecuciones distintas.
  2. Las observaciones provienen de una distribucin cercana a la normal: Para verificar esto se realiza una prueba Shapiro-Wilk y un grfico Q-Q para normalidad.

En estos casos se puede usar la prueba de suma de rangos de Wilcoxon, que es una alternativa no paramtrica para t-Student de muestras independientes.

Para realizar el anlisis estadstico debemos enunciar nuestras hiptesis:

\(H_0:\) No hay diferencias significativas en la eficiencia entre las versiones A y B del algoritmo cuando las instancias tienen 70 o ms nodos.

\(H_A:\) Hay diferencias significativas en la eficiencia entre las versiones A y B del algoritmo cuando las instancias tienen 70 o ms nodos.

Luego verificamos las condiciones para esta prueba:

  1. Observaciones de ambas muestras deben ser independientes: Las muestras fueron seleccionadas aleatoriamente e independientes entre ellas.

  2. Escala de medicin a lo menos ordinal: Como la variable que se est midiendo es el tiempo, que cumple con las caractersticas de una escala de razn, se puede concluir que tambin cumple con ser ordinal.

A continuacin, realizamos la prueba de suma de rangos de Wilcoxon para concluir sobre las hiptesis enunciadas.

resultWilcoxSum <- wilcox.test(muestraTiempos_A, muestraTiempos_B, paired = FALSE, alternative = "two.sided", 0, 0.95)

Con lo anterior, se observa que si existen diferencias significativas en los tiempos de ejecucin de los algoritmos A y B.
